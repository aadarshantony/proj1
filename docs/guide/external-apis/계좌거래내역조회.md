## 1. 개요

### 1.1 기업계좌 조회 API란?

Hyphen의 기업계좌 조회 API는 법인/개인사업자의 은행 계좌 정보를 조회할 수 있는 스크래핑 기반 API입니다. 인터넷/기업뱅킹 인증 정보를 이용하여 계좌 잔액, 거래내역, 보유 계좌 목록 등을 조회할 수 있습니다.

```
[조회 가능 정보]
- 보유 계좌 목록 (전계좌조회)
- 계좌별 실시간 잔액 및 출금가능액
- 입출금 거래내역 (거래일시, 금액, 거래처명, 적요)
- 이체 결과 확인
```

### 1.2 Hyphen 기업계좌 조회 API

[Hyphen](https://hyphen.im)의 기업계좌 조회 API를 통해 다양한 은행의 계좌 정보를 통합 조회할 수 있습니다.

- **API 문서**: https://hyphen.im/product-api/view?seq=54
- **지원 은행**: 국내 주요 은행 20개+ (1금융권, 2금융권 포함)
- **인증 보안**: ISMS 정보보호 관리체계 인증

### 1.3 SaaS 관리 플랫폼에서의 활용

| 활용 목적          | 설명                                         |
| ------------------ | -------------------------------------------- |
| **SaaS 결제 추적** | 계좌이체로 결제된 SaaS 구독료 내역 자동 수집 |
| **자금 현황 파악** | 실시간 계좌 잔액으로 현금흐름 모니터링       |
| **결제 이력 통합** | 법인카드 + 계좌이체 결제 통합 관리           |
| **비용 분석**      | 거래처명 기반 SaaS 서비스별 비용 집계        |
| **이상 거래 탐지** | 예상치 못한 출금 내역 알림                   |

### 1.4 제공 API 목록

## 2. 사전 준비

### 2.1 Hyphen 회원가입 및 상품 신청

1. [Hyphen 회원가입](https://hyphen.im/user/sign-up)
2. 로그인 후 "기업계좌 조회" 상품 신청
3. 인증 정보 발급 (이메일로 Hkey 전달)

### 2.2 요금제 선택

| 요금제          | 특징                  | 결제 방식                  |
| --------------- | --------------------- | -------------------------- |
| **TR 슬림**     | 계약서 없이 즉시 사용 | 비즈머니 충전 후 선불 차감 |
| **TR 시그니처** | 계약서 필요           | 후불 청구                  |

> **테스트**: 일 100건, 월 100건 무료 테스트 제공

### 2.3 인증 정보 준비

```
user_id: 회원가입 시 생성된 ID
hkey: 회원가입 후 이메일로 전달받은 보안키
```

### 2.4 은행 인증 정보

기업계좌 조회를 위해서는 해당 은행의 인터넷/기업뱅킹 인증 정보가 필요합니다.

#### ID 로그인 방식

```typescript
interface IdLoginCredential {
  loginType: "ID"; // 로그인 유형
  userId: string; // 은행 사이트 ID
  userPw: string; // 은행 사이트 비밀번호
}
```

#### 인증서 로그인 방식

```typescript
interface CertLoginCredential {
  loginType: "CERT"; // 로그인 유형
```

## 3. 인증 (OAuth 2.0)

### 3.1 Access Token 발급

**Endpoint**

```
POST https://api.hyphen.im/oauth/token
```

**Request**

```typescript
// Header
{ "Content-Type": "application/json" }

// Body
{
  "user_id": "your_user_id",
  "hkey": "your_hkey"
}
```

**Response**

```typescript
{
  "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6...",
  "expiresIn": 604800  // 7일 (초 단위)
}
```

### 3.2 토큰 관리

```typescript
// lib/hyphen-auth.ts
interface TokenInfo {
  accessToken: string;
  expiresAt: Date;
}

let tokenCache: TokenInfo | null = null;

export async function getAccessToken(): Promise<string> {
  // 토큰이 없거나 만료 임박 시 재발급
  if (!tokenCache || isExpiringSoon(tokenCache.expiresAt)) {
    tokenCache = await refreshToken();
  }
  return tokenCache.accessToken;
}

function isExpiringSoon(expiresAt: Date): boolean {
  const buffer = 60 * 60 * 1000; // 1시간 전 미리 갱신
  return Date.now() > expiresAt.getTime() - buffer;
}

async function refreshToken(): Promise<TokenInfo> {
  const response = await fetch("https://api.hyphen.im/oauth/token", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      user_id: process.env.HYPHEN_USER_ID,
      hkey: process.env.HYPHEN_HKEY,
    }),
  });

  const data = await response.json();
  return {
    accessToken: data.accessToken,
    expiresAt: new Date(Date.now() + data.expiresIn * 1000),
  };
}
```

### 3.3 API 호출 시 인증 헤더


## 4. API 명세

### 4.1 기업계좌 잔액조회

계좌의 현재 잔액과 출금가능액을 조회합니다.

**Endpoint**

```
POST https://api.hyphen.im/in0087000482
```

**Request**

```typescript
interface BalanceRequest {
  // 은행 인증 정보
  bankCd: string; // 은행코드 (예: "004" 국민은행)
  loginType: "ID" | "CERT"; // 로그인 방식
  userId?: string; // ID 로그인 시
  userPw?: string; // ID 로그인 시
  signCert?: string; // 인증서 로그인 시
  signPri?: string; // 인증서 로그인 시
  signPw?: string; // 인증서 로그인 시

  // 조회 정보
  acctNo: string; // 계좌번호 (숫자만)
}
```

**Response**

```typescript
interface BalanceResponse {
  code: string; // 결과코드 ("00000": 성공)
  message: string; // 결과메시지
  data: {
    bankCd: string; // 은행코드
    acctNo: string; // 계좌번호
    acctNm: string; // 계좌명
    balance: string; // 현재잔액
    availableAmt: string; // 출금가능액
    lastTrDt: string; // 최종거래일 (YYYYMMDD)
    lastTrTm: string; // 최종거래시간 (HHmmss)
  };
}
```

**예제**

```typescript
// 잔액 조회 함수
async function getAccountBalance(
  bankCd: string,
  acctNo: string,
  credential: IdLoginCredential | CertLoginCredential
): Promise<BalanceResponse> {
  const accessToken = await getAccessToken();

  const response = await fetch("https://api.hyphen.im/in0087000482", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      Authorization: `Bearer ${accessToken}`,
    },
    body: JSON.stringify({
      bankCd,
      acctNo,
      ...credential,
    }),
  });

  return response.json();
}

// 사용 예시
const balance = await getAccountBalance("004", "12345678901234", {
  loginType: "ID",
  userId: "bank_user",
  userPw: "bank_password",
});

console.log(`잔액: ${Number(balance.data.balance).toLocaleString()}원`);
console.log(
  `출금가능액: ${Number(balance.data.availableAmt).toLocaleString()}원`
);
```

### 4.2 기업계좌 거래내역조회 (핵심)

지정 기간의 입출금 거래내역을 조회합니다. SaaS 결제 추적의 핵심 API입니다.

**Endpoint**

```
POST https://api.hyphen.im/in0087000483
```

**Request**

```typescript
interface TransactionRequest {
  // 은행 인증 정보
  bankCd: string; // 은행코드
  loginType: "ID" | "CERT"; // 로그인 방식
  userId?: string; // ID 로그인 시
  userPw?: string; // ID 로그인 시
  signCert?: string; // 인증서 로그인 시
  signPri?: string; // 인증서 로그인 시
  signPw?: string; // 인증서 로그인 시

  // 조회 조건
  acctNo: string; // 계좌번호
  startDt: string; // 조회시작일 (YYYYMMDD)
  endDt: string; // 조회종료일 (YYYYMMDD)
  trType?: string; // 거래유형 ("A": 전체, "I": 입금, "O": 출금)
  pageNo?: string; // 페이지번호
  pageSize?: string; // 페이지크기
}
```

**Response**

```typescript
interface TransactionResponse {
  code: string; // 결과코드 ("00000": 성공)
  message: string; // 결과메시지
  data: {
    totalCount: string; // 전체 건수
    list: TransactionItem[];
  };
}

interface TransactionItem {
  trDt: string; // 거래일자 (YYYYMMDD)
  trTm: string; // 거래시간 (HHmmss)
  trNo: string; // 거래번호
  trType: string; // 거래유형 ("I": 입금, "O": 출금)
  inAmt: string; // 입금금액
  outAmt: string; // 출금금액
  balance: string; // 거래후잔액
  trNm: string; // 거래처명 (입금자/수취인)
  memo: string; // 적요
  branchNm?: string; // 거래점명
  recvAcctNo?: string; // 상대계좌번호
  recvAcctHolder?: string; // 상대예금주명
}
```

**예제**

```typescript
// 거래내역 조회 함수
async function getTransactions(
  bankCd: string,
  acctNo: string,
  startDt: string,
  endDt: string,
  credential: IdLoginCredential | CertLoginCredential,
  options?: { trType?: string; pageNo?: string; pageSize?: string }
): Promise<TransactionResponse> {
  const accessToken = await getAccessToken();

  const response = await fetch("https://api.hyphen.im/in0087000483", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      Authorization: `Bearer ${accessToken}`,
    },
    body: JSON.stringify({
      bankCd,
      acctNo,
      startDt,
      endDt,
      trType: options?.trType || "A",
      pageNo: options?.pageNo || "1",
      pageSize: options?.pageSize || "100",
      ...credential,
    }),
  });

  return response.json();
}

// 사용 예시: 이번 달 출금 내역 조회
const today = new Date();
const startDt = `${today.getFullYear()}${String(today.getMonth() + 1).padStart(2, "0")}01`;
const endDt = `${today.getFullYear()}${String(today.getMonth() + 1).padStart(2, "0")}${String(today.getDate()).padStart(2, "0")}`;

const transactions = await getTransactions(
  "004",
  "12345678901234",
  startDt,
  endDt,
  { loginType: "ID", userId: "bank_user", userPw: "bank_password" },
  { trType: "O" } // 출금만
);

// 거래 내역 출력
transactions.data.list.forEach((tr) => {
  console.log(
    `${tr.trDt} ${tr.trTm} | ${tr.trNm} | -${Number(tr.outAmt).toLocaleString()}원`
  );
});
```

```typescript
// SaaS 서비스 패턴 정의
const SAAS_PATTERNS = [
  { pattern: /AWS|AMAZON WEB SERVICES/i, service: "AWS" },

  { pattern: /SALESFORCE/i, service: "Salesforce" },
  { pattern: /HUBSPOT/i, service: "HubSpot" },
  { pattern: /ZENDESK/i, service: "Zendesk" },
  { pattern: /INTERCOM/i, service: "Intercom" },
  { pattern: /DATADOG/i, service: "Datadog" },
  { pattern: /SENDGRID/i, service: "SendGrid" },
  { pattern: /TWILIO/i, service: "Twilio" },
  { pattern: /VERCEL/i, service: "Vercel" },
  { pattern: /NETLIFY/i, service: "Netlify" },
  { pattern: /HEROKU/i, service: "Heroku" },
];

interface DetectedSaaSPayment {
  transactionDate: string;
  transactionTime: string;
  serviceName: string;
  amount: number;
  originalTransactionName: string;
  memo: string;
}

export function detectSaaSPayments(
  transactions: TransactionItem[]
): DetectedSaaSPayment[] {
  const detected: DetectedSaaSPayment[] = [];

  for (const tr of transactions) {
    // 출금 거래만 분석
    if (tr.trType !== "O" || Number(tr.outAmt) === 0) continue;

    // 거래처명 또는 적요에서 SaaS 패턴 매칭
    const searchText = `${tr.trNm} ${tr.memo}`;

    for (const { pattern, service } of SAAS_PATTERNS) {
      if (pattern.test(searchText)) {
        detected.push({
          transactionDate: tr.trDt,
          transactionTime: tr.trTm,
          serviceName: service,
          amount: Number(tr.outAmt),
          originalTransactionName: tr.trNm,
          memo: tr.memo,
        });
        break; // 첫 번째 매칭만 사용
      }
    }
  }

  return detected;
}
```

```typescript
interface UnifiedPayment {
  date: string;
  serviceName: string;
  amount: number;
  paymentMethod: "CARD" | "TRANSFER";
  cardNo?: string;
  accountNo?: string;
  description: string;

  totalAmount: number;
  cardAmount: number;
  transferAmount: number;
  byService: Record<string, number>;
  payments: UnifiedPayment[];

export async function getUnifiedPaymentSummary(
  organizationId: string,
  startDt: string,
  endDt: string
): Promise<PaymentSummary> {
  // 1. 법인카드 사용내역 조회 (법인카드-사용내역.md 참조)
  const cardPayments = await getCardSaaSPayments(
    organizationId,
    startDt,
    endDt
  );

  // 2. 계좌이체 내역 조회
  const accountPayments = await getAccountSaaSPayments(
    organizationId,
    startDt,
    endDt
  );

  // 3. 통합 정리
  const payments: UnifiedPayment[] = [
    ...cardPayments.map((p) => ({
      date: p.approvalDate,
      serviceName: p.serviceName,
      amount: p.amount,
      paymentMethod: "CARD" as const,
      cardNo: p.cardNo,
      description: p.merchantName,
    })),
    ...accountPayments.map((p) => ({
      date: p.transactionDate,
      serviceName: p.serviceName,
      amount: p.amount,
      paymentMethod: "TRANSFER" as const,
      accountNo: p.accountNo,
      description: p.originalTransactionName,
    })),
  ];

  // 4. 집계
  const byService: Record<string, number> = {};
  let cardAmount = 0;
  let transferAmount = 0;

  for (const p of payments) {
    byService[p.serviceName] = (byService[p.serviceName] || 0) + p.amount;
    if (p.paymentMethod === "CARD") {
      cardAmount += p.amount;
    } else {
      transferAmount += p.amount;
    }
  }

  return {
    totalAmount: cardAmount + transferAmount,
    cardAmount,
    transferAmount,
    byService,
    payments: payments.sort((a, b) => b.date.localeCompare(a.date)),
```

### 6.3 현금흐름 모니터링

실시간 계좌 잔액과 일별 입출금 추이를 모니터링합니다.

```typescript
// lib/services/cashflow-monitor.ts

interface DailyCashflow {
  date: string;
  inflow: number; // 입금 합계
  outflow: number; // 출금 합계
  netflow: number; // 순유입 (입금 - 출금)
  balance: number; // 잔액
}

interface CashflowReport {
  period: { start: string; end: string };
  accounts: AccountCashflow[];
  totalBalance: number;
  totalInflow: number;
  totalOutflow: number;
  dailyTrend: DailyCashflow[];
}

interface AccountCashflow {
  bankCd: string;
  bankNm: string;
  acctNo: string;
  acctNm: string;
  currentBalance: number;
  periodInflow: number;
  periodOutflow: number;
}

export async function generateCashflowReport(
  organizationId: string,
  startDt: string,
  endDt: string
): Promise<CashflowReport> {
  // 조직의 등록된 모든 계좌 조회
  const registeredAccounts = await getRegisteredAccounts(organizationId);

  const accounts: AccountCashflow[] = [];
  const dailyMap = new Map<string, DailyCashflow>();
  let totalBalance = 0;
  let totalInflow = 0;
  let totalOutflow = 0;

  for (const account of registeredAccounts) {
    // 잔액 조회
    const balance = await getAccountBalance(
      account.bankCd,
      account.acctNo,
      account.credential
    );

    // 거래내역 조회
    const transactions = await getTransactions(
      account.bankCd,
      account.acctNo,
      startDt,
      endDt,
      account.credential
    );

    let periodInflow = 0;
    let periodOutflow = 0;

    for (const tr of transactions.data.list) {
      const inAmt = Number(tr.inAmt) || 0;
      const outAmt = Number(tr.outAmt) || 0;

      periodInflow += inAmt;
      periodOutflow += outAmt;

      // 일별 집계
      const daily = dailyMap.get(tr.trDt) || {
        date: tr.trDt,
        inflow: 0,
        outflow: 0,
        netflow: 0,
        balance: 0,
      };
      daily.inflow += inAmt;
      daily.outflow += outAmt;
      daily.netflow = daily.inflow - daily.outflow;
      dailyMap.set(tr.trDt, daily);
    }

    const currentBalance = Number(balance.data.balance);
    accounts.push({
      bankCd: account.bankCd,
      bankNm: BANK_NAMES[account.bankCd],
      acctNo: account.acctNo,
      acctNm: balance.data.acctNm,
      currentBalance,
      periodInflow,
      periodOutflow,
    });

    totalBalance += currentBalance;
    totalInflow += periodInflow;
    totalOutflow += periodOutflow;
  }

  // 일별 추이 정렬
  const dailyTrend = Array.from(dailyMap.values()).sort((a, b) =>
    a.date.localeCompare(b.date)
  );

  return {
    period: { start: startDt, end: endDt },
    accounts,
    totalBalance,
    totalInflow,
    totalOutflow,
    dailyTrend,
  };
}
```

### 6.4 Server Action 예제

Next.js Server Action으로 거래내역 조회를 구현합니다.

```typescript
// actions/account.ts
"use server";

import { z } from "zod";
import { auth } from "@/lib/auth";
import { prisma } from "@/lib/db";
import { revalidatePath } from "next/cache";

const transactionQuerySchema = z.object({
  accountId: z.string(),
  startDate: z.string().regex(/^\d{8}$/),
  endDate: z.string().regex(/^\d{8}$/),
  transactionType: z.enum(["A", "I", "O"]).default("A"),
});

interface TransactionQueryResult {
  success: boolean;
  data?: {
    transactions: TransactionItem[];
    totalCount: number;
    summary: {
      totalInflow: number;
      totalOutflow: number;
      netflow: number;
    };
  };
  error?: string;
}

export async function queryAccountTransactions(
  input: z.infer<typeof transactionQuerySchema>
): Promise<TransactionQueryResult> {
  try {
    const session = await auth();
    if (!session?.user?.organizationId) {
      return { success: false, error: "인증이 필요합니다" };
    }

    const validated = transactionQuerySchema.parse(input);

    // 계좌 정보 조회 (조직 소속 확인)
    const account = await prisma.bankAccount.findFirst({
      where: {
        id: validated.accountId,
        organizationId: session.user.organizationId,
      },
    });

    if (!account) {
      return { success: false, error: "계좌를 찾을 수 없습니다" };
    }

    // 은행 인증 정보 복호화
    const credential = await decryptCredential(account.encryptedCredential);

    // Hyphen API 호출
    const response = await getTransactions(
      account.bankCd,
      account.acctNo,
      validated.startDate,
      validated.endDate,
      credential,
      { trType: validated.transactionType }
    );

    if (response.code !== "00000") {
      return { success: false, error: response.message };
    }

    // 집계 계산
    let totalInflow = 0;
    let totalOutflow = 0;

    for (const tr of response.data.list) {
      totalInflow += Number(tr.inAmt) || 0;
      totalOutflow += Number(tr.outAmt) || 0;
    }

    return {
      success: true,
      data: {
        transactions: response.data.list,
        totalCount: Number(response.data.totalCount),
        summary: {
          totalInflow,
          totalOutflow,
          netflow: totalInflow - totalOutflow,
        },
      },
    };
  } catch (error) {
    console.error("계좌 거래내역 조회 실패:", error);
    return { success: false, error: "거래내역 조회에 실패했습니다" };
  }
}
```

### 7.3 에러 핸들링 예제

```typescript
// lib/hyphen-error-handler.ts

class HyphenApiError extends Error {
  constructor(
    public code: string,
    message: string,
    public isRetryable: boolean = false
  ) {
    super(message);
    this.name = "HyphenApiError";
  }
}

const RETRYABLE_ERRORS = ["40001", "40002"];

export async function withRetry<T>(
  operation: () => Promise<T>,
  maxRetries: number = 3,
  delayMs: number = 1000
): Promise<T> {
  let lastError: Error | null = null;

  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      const result = await operation();

      // 응답 코드 확인
      if (typeof result === "object" && result !== null && "code" in result) {
        const response = result as { code: string; message: string };

        if (response.code !== "00000") {
          const isRetryable = RETRYABLE_ERRORS.includes(response.code);

          if (isRetryable && attempt < maxRetries) {
            await delay(delayMs * attempt); // 지수 백오프
            continue;
          }

          throw new HyphenApiError(
            response.code,
            response.message,
            isRetryable
          );
        }
      }

      return result;
    } catch (error) {
      lastError = error as Error;

      if (error instanceof HyphenApiError && !error.isRetryable) {
        throw error;
      }

      if (attempt < maxRetries) {
        await delay(delayMs * attempt);
      }
    }
  }

  throw lastError || new Error("알 수 없는 오류");
}

function delay(ms: number): Promise<void> {
  return new Promise((resolve) => setTimeout(resolve, ms));
}

// 사용 예시
const result = await withRetry(
  () => getTransactions(bankCd, acctNo, startDt, endDt, credential),
  3,
  2000
);
```
